Feature log
April 16, 2026
Rain delay & game status locking
Picks now lock during rain delays, suspensions, warmups, and pre-game — not just when the game is live. The lock banner also shows the specific reason (e.g. "game is delayed") instead of a generic message.
April 16, 2026
User accounts & login
Each player now has their own account with a username and password. Picks are tied to your account — you can only make a pick for yourself. Other players' picks are hidden until the game starts so nobody can copy anyone.
April 16, 2026
Game-time pick locking
Picks are now editable right up until first pitch. No more manual "lock" button — the app detects when the game starts via the MLB API and freezes picks automatically.
April 16, 2026
Shared database via Supabase
All picks, scores, and accounts are now stored in a shared database. Everyone in the league sees the same leaderboard and picks in real time from any device.
April 16, 2026
Live MLB stats integration
No more manual stat entry. The app pulls live boxscore data directly from the official MLB Stats API after each game — hits, doubles, triples, home runs, RBIs, runs, walks, stolen bases, strikeouts, and pitching stats are all automatic.
April 16, 2026
Deployed to the web
Pick the Stick is now a live website hosted on Netlify. Share the link with your friends to play from any device — no install required.
April 16, 2026
Pick locking (manual)
Added a "Save & lock picks" button so picks couldn't be changed after everyone had submitted. Validated that all players had a pick before allowing the lock.
April 16, 2026
Scoring rules reference
Added a Rules tab with the full 2026 Fieger Law scoring breakdown — offense, pitching, penalties, and the special lineup-doubler rule.
April 16, 2026
Season leaderboard
Added a leaderboard tab tracking cumulative season points for all players with medal rankings.
April 16, 2026
Pick the Stick launched 🐯
First version of the app built — daily player picker for the Detroit Tigers, point scoring based on Fieger Law rules, and a basic leaderboard. The project was born.
